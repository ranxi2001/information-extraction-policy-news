海南省人民政府关于积极发挥新消费引领作用加快培育形成新供给新动力的实施方案
为贯彻落实《国务院关于积极发挥新消费引领作用加快培育形成新供给新动力的指导意见》(国发〔2015〕66号)精神，结合我省实际，制定本方案。
一、总体要求 
(一)指导思想。全面贯彻党的十八大和十八届历次全会和习近平总书记系列重要讲话及省第七次党代会精神，按照国务院及省委、省政府的决策部署，围绕省第七次党代会提出“建设美好新海南”奋斗目标，针对新消费需求不断扩大、新消费结构不断调整升级、传统供给难以满足需求、消费增速不快等问题，主动引领新消费、适应消费新常态。以满足海南国际旅游岛消费升级需求作为总抓手，以深入推进供给侧结构性改革为主线，增加海南特色消费产品有效供给，为建设美好新海南提供更持续、更强劲的新动力。 
(二)主要目标。重点发挥旅游、医疗健康、特色高效农业、信息、教育、文化体育等六个领域新消费引领作用，培育形成海南未来一批“专、精、特、新”消费产品，以满足日益多元化、个性化消费需求为目标，加快完善新产品设计，扩大有效供给，优化消费环境，着力破解与新消费需求不相适应、有效供给不足的问题，推动消费平稳较快增长和结构优化。到2020年，全省社会消费品零售总额达到1940亿元，年均增长8%以上，力争有更快增长。
二、创新有效供给，推动重点领域消费发展 
(一)构建海南特色旅游产业体系，满足旅游消费升级需求。
全面创建“全域旅游示范省”，创新旅游业态，深度开发富有魅力的度假休闲旅游产品和精品线路，满足旅游消费升级需求。
1.提升旅游基础设施和公共服务水平。完善交通基础设施。加快建设环岛旅游公路，完成海口美兰机场和三亚凤凰机场改扩建，推动建设三亚新机场，打通机场、公路、高铁至各景区景点的连接通道 ;建设城市、滨海休闲道路，增加自行车道、轮滑道、步行道等休闲交通设施，为游客提供更加便捷、高效、多样化的旅游交通服务。继续加大国际航线开辟力度。积极开辟海南至德国、日本、澳大利亚等境外新航线 ;推动 59国公民来琼免签政策落地实施。提升旅游公共服务能力。加快旅游咨询中心和集散中心、旅游厕所、停车场、充电桩等建设，提升旅游标识和旅游服务标准化水平。加快旅游要素和旅游设施国际化改造，提高外语交流、外币兑换等国际化旅游公共服务能力。积极发展海南 “互联网 +旅游 ”服务支撑系统。 (责任单位：省发展改革委、省交通运输厅、省旅游委、省住房城乡建设厅、省公安厅、省政府金融办、省民航办、海南出入境检验检疫局 ) 
2.培育壮大海洋旅游消费。做大做强邮轮旅游。建成三亚、海口邮轮母港，完善口岸及旅游相关配套设施，增强邮轮母港的综合保障服务能力。加快三沙邮轮停靠港建设。积极开发连接东北亚、东南亚、港澳台地区的区域性和国际性邮轮航线，争取开辟经西沙、南沙至新加坡、马来西亚、印度尼西亚、文莱等国际邮轮旅游航线。常态化运营西沙邮轮旅游，优化旅游航线，打造西沙旅游精品，适时开发环海南岛邮轮航线和南海邮轮旅游航线。积极开发游艇旅游。培育游艇大众消费市场，简化审批手续，放宽航行条件，进一步开放岸线和水域，降低境外游艇入境门槛，大力推进境外游艇自由行 ;积极申请成为国家游艇租赁试点省份，取消入境游艇访问担保金 ;高水平建设秀英港、金沙湾、半山半岛、鸿洲等游艇码头 ;重点推进公共游艇码头和基础配套设施建设 ;鼓励开展游艇租赁服务业务，大力发展游艇赛事会展商务活动。开发海洋旅游新业态。加快建设海洋公园，开展海钓、远航、潜水等多样化的海上旅游项目。积极发展海底观光游、岛礁体验游、渔村风情小镇游、休闲渔业游、海上游船美食游等海洋旅游新业态。 (责任单位：省旅游委、省发展改革委、省交通运输厅、省海洋与渔业厅、省海防与口岸办、海南出入境检验检疫局，海口、三亚、三沙等市县政府 ) 
3.培育壮大房车露营、低空飞行、婚庆旅游等旅游新业态。发展房车露营旅游。完成全岛自驾车、房车露营地的节点状空间布局，依托重要景区和度假区建设一批标准化自驾车、房车露营地，打造环岛房车露营旅游线。发展低空飞行旅游。充分利用海南作为低空政策开放的先行试点优势，加强通航机场及配套设施建设，打造海口、三亚、三沙三大低空飞行试点基地，优化低空飞行管制服务，推动构建通用航空政策法规体系。支持常驻通航企业开通覆盖全岛的低空旅游航线。发展和丰富婚庆旅游产品。打造天涯海角、海口观澜湖等 12个特色婚庆基地，塑造婚庆蜜月旅游品牌。 (责任单位 :省旅游委、省民航办，有关市县政府 ) 
4.培育壮大乡村旅游消费。推进美丽海南百镇千村建设。以宜居宜业宜游为目标，充分挖掘、利用本地资源环境优势，提升镇村公共设施、服务和管理水平，强化完善观光、体验等旅游功能，将百个特色产业小镇和千个美丽乡村打造成为全域旅游重要支撑点。打造精品乡村旅游品牌。重点培育 “北仍村 ”“百里百村 ”等乡村旅游品牌，开发系列乡村生态旅游路线 ;重点打造三亚玫瑰谷、三亚海棠湾水稻公园、保亭槟榔谷、琼海龙寿洋、五指山牙胡梯田、文昌龙楼好圣航天科技小康村等精品景点。大力发展休闲农业，培育壮大经营主体，引导农家乐、渔家乐等走多样化、特色化、精品化路线，丰富乡村旅游产品供给。举办乡村旅游节庆活动。办好一年一度的 “海南乡村旅游文化节 ”，探索 “欢乐节 ”与“冬交会 ”联办或合办，扩大海南乡村旅游影响。 (责任单位：省旅游委、省农业厅、省住房城乡建设厅，各市县政府 ) 
5.培育壮大购物旅游消费。进一步做强做优离岛免税购物。在离岛免税购物政策方面，争取提高单件商品免税限额，推动增加免税购物网点。积极对接消费热点，优化免税商品结构。大力发展 “互联网 +免税购物 ”，丰富免税购物方式、拓宽免税购物渠道。增加特色旅游商品供给。挖掘发展黎锦苗绣、椰壳贝艺等传统手工技艺，打造具有海南特色和文化底蕴的旅游商品，建设好 “海南礼物 ”旅游商品平台，全面提升旅游商品的研发、设计、生产、推广、销售的水平和规模。举办创意旅游商品设计大赛，推出一批特色旅游精品，将旅游商品打造成海南特色品牌。构建多元旅游购物体验场所，以免税购物店、休闲购物街、旅游商品专卖店、景区商店、乡村连锁店五类线下实体店和一个线上旅游商店为主体，打造 “5+1”特色旅游购物载体，多维促进旅游购物消费。丰富购物促销活动。定期举办 “海南国际旅游岛购物节 ”、海南名特优产品采购大会。支持大型购物中心、龙头企业、行业协会或商会，引进国内外知名时尚企业、时尚品牌和时尚产品，举办进口商品展、时装周等系列活动，营造良好的消费氛围。 (责任单位：省财政厅、省商务厅、省旅游委、海南出入境检验检疫局 ) 
6.培育壮大产业旅游和旅游装备制造。推进航天科技旅游。打造航天主题公园，配套建设航天主题度假酒店、餐厅、商店，推出航天科技和太空文化系列旅游产品和主题商品，塑造航天旅游品牌。发展产业园区旅游。探索向信息产业园、动漫软件园、农产品加工园注入旅游元素，打造集参观、科普、体验为一体的旅游产品和线路。发展旅游装备制造。重点发展游艇、轻型水上飞机、房车、越野和户外运动装备、潜水设备、高尔夫用具等旅游装备制造业。 (责任单位：省旅游委、省发展改革委、省工业和信息化厅、省商务厅，各有关市县政府 )
到2020年，全年接待旅游总人数超过 8000万人次，年均增长 8.5%;接待入境游客达到 120万人次以上 ;旅游总收入达到 1000亿元以上，年均增长 11.8%;过夜游客人均停留 4.2天，过夜游客人均消费 3500元。 
(二)培育发展医疗健康产业，满足健康消费需求。
按照建设健康岛、长寿岛的要求，推动医疗健康产业加快发展，增加业态丰富、结构合理、多元多层次的健康消费供给，满足健康消费需求。 
7.充分释放博鳌乐城国际医疗旅游先行区 (以下简称先行区 )的优惠政策效应，将先行区打造成国际化医疗技术服务产业集聚区。继续试行 “极简审批 ”，全面落实细化国务院赋予先行区的九大优惠政策，加快基础设施建设和提升公共服务水平。推动博鳌麦迪赛尔国际医疗中心、中国干细胞集团附属医院等一批在建项目尽快建成使用，继续引进和建设一批新的高端医疗养生、健康管理、康复护理项目，形成体检、健康管理、医疗服务、康复、养生等完整的医疗健康消费链。依托先行区医疗机构，积极建设国家级重点专科和实验室，开展医学科研和先进技术引进，推动先行区产学研融合发展。精品化开发医疗旅游项目和产品，推动与旅游业融合发展。 (责任单位：省卫生计生委、省发展改革委、省旅游委、海南出入境检验检疫局，琼海市政府 ) 
8.加快一批医疗健康养生项目落地。加强与国内外企业和先进医疗机构合作，大力引进优质医疗资源合作办医，打造一批高端医疗、健康服务项目。加快建设兴隆医疗健康养生基地。培育发展健康孕育一体化产业链，支持高端优质妇产专科医院建设。鼓励社会资本投资健康体检、健康咨询、健康文化、中医健康养生等多样化的健康产业。精心策划一批健康旅游项目，满足岛外医疗健康消费需求。 (责任单位：省卫生计生委、省发展改革委，琼海市政府 ) 
9.发展中医养生康复疗养。推进三亚和海口中医医疗旅游示范基地建设，引入国际医疗卫生机构认证，形成高端医疗旅游的产业配套和服务体系。打造海南温泉休闲生活方式，坚持走 “温泉 +”的产业集聚开发之路，推出“冬季温泉 ”“夏季冷泉 ”等康体美容与养生产品，塑造热带海岛温泉度假品牌。 (责任单位：省卫生计生委、省旅游委，海口、三亚、琼海、定安等市县政府 ) 
10.鼓励社会资本发展养老服务。推进高端养老服务公办民营、民办公助、政府和社会资本合作共建，支持境外资本投资高端养老服务业，在海口、三亚、儋州等城市开展医养结合试点示范工作。 (省民政厅、省卫生计生委，海口、三亚、儋州等市县政府 )
到2020年，全省新增各类养老床位 1.03万张，机构养老床位达到 4.39万张，医疗健康产业增加值占全省地区
生产总值的比重达到 5%以上，医疗健康产业总产值达到 1000亿元。
(三)〖JP3〗加快发展热带特色高效农业，对接优质农产品消费需求。
以“把热带特色高效农业打造成为富足农民、服务全国的王牌产业 ”为目标，以市场需求为导向，增加优质农产品有效供给，满足优质农产品消费需求。 
11.推进农业品牌建设。围绕全省特色优势农产品，以龙头企业为重点，加大品牌创建与培育力度，积极引导注册并依法规范使用农产品商标以及地理标志商标，创建扶持一批品牌价值高、综合竞争力强的自主品牌。培育壮大椰树、椰岛、春光、南国等本土品牌。加强品牌保护与推介，健全农业商标品牌建设工作机制。推广澄迈 “商标富农 ”经验，推行 “公司 +商标品牌 (地理标志 )+农户 ”产业化经营模式，提高农产品竞争力。以 “冬交会”等展会为平台，开展以环渤海、长三角、珠三角三大地区和境外为重点市场的系列促销活动，扩大海南农业品牌的影响力。 (责任单位：省农业厅、省工商局，各市县政府 ) 
12.增加优质农产品有效供给。发展优质高效种植业。按照 “强种、优菜、扩果、稳作 ”的思路，调整优化种植结构，提高优质、高效益农产品供给。发展集约化育苗，做精做优瓜菜产业。积极发展热带水果，建设一批规模较大热带水果基地，增加热带特色优质水果供给。大力发展槟榔深加工，适度发展胡椒、椰子，逐步恢复咖啡、腰果等本地热带作物。调整优化养殖业结构，针对市场需求，按照 “稳猪、促禽、增牛羊 ”的思路，调整优化养殖结构。重点发展黑猪、文昌鸡等特色畜禽养殖业，增加牛、羊等畜牧产品有效供给 ;引进澳洲优良肉牛品种，为市场提供高端优牛肉产品。 (责任单位：省农业厅，各市县政府 )
到2020年，冬季瓜菜种植面积稳定在 300万亩，产量达到 495万吨，出岛量达到 370万吨 ;热带水果种植面积稳定在 270万亩左右，总产达到 450万吨 ;家禽总出栏 1.8亿只，禽肉产量 24.5万吨，禽蛋产量 5万吨，牛出栏 35万头、羊出栏 105万只。 
13.发展休闲农业。挖掘民族村镇、民风民俗文化，由初级观光休闲向创意体验转变，由分散经营向规模化、组团式、线路化发展转变。建设一批集科普、教育、示范功能于一体的休闲农庄，支持经营主体联合打造一批精品休闲农业线路、特色产业带和产业集群，促进休闲观光农业提档升级。积极参与 “国家田园综合体试点”，开展 “共享海南农庄 ”行动。培育桂林洋农业公园、海南陵水现代农业示范基地等一批休闲农业品牌。 (责任单位：省农业厅、省旅游委，各市县政府 )
到2020年，打造海南农庄 1000家，创建 “全国休闲农业与乡村旅游示范点 ”20个，省级休闲农业示范点达到 200个。 
14.推进 “互联网 +农业 ”。按照省域 “多规合一 ”和百个特色产业小镇规划统筹规划全省互联网农业小镇，培育建设一批农业电子商务平台，支持发展农产品在线交易，建设集农业生产、农产品加工销售、农旅融合、民俗风情、康体养生为一体的互联网小镇。依托海南省政务大数据服务平台建立农业大数据分析系统，按照统一数据标准规范管理全省农业数据。鼓励扶持农业企业、专业合作社大力发展农村电子商务。 (责任单位：省农业厅、省工业和信息化厅、省邮政管理局、省供销合作社，各市县政府 )
到2020年，初步构建 “一库一中心，一图一平台，两网 N应用 ”的海南农业大数据服务平台体系，实现全省农业数据共享。农产品电子商务交易额突破 200亿元，涉农企业电子商务应用率达 30%。 
(四)加快发展互联网产业，打造信息消费新热点。
顺应互联网时代潮流，提升信息化基础设施条件，增加新型信息服务产品供给，打造信息消费新热点。 
15.营造良好信息消费环境。适应消费结构、消费模式和消费形态变化，加快信息化、网络化、智能化基础设施建设，提高 “三个全覆盖 ”(光纤宽带网络和第四代移动通信网络在城市和行政村全覆盖、重点公共场所无线网络全覆盖 )水平。前瞻性研究布局第五代移动通信网络。进一步推动网络 “提速降费 ”。取消国内漫游费，营造良好的信息消费环境。 (责任单位：省工业和信息化厅、省通信管理局、省文化广电出版体育厅，各市县政府 )
到2020年，基本建成覆盖城乡、服务便捷、高速通畅、技术先进的宽带网络基础设施。全省有线电视网络互联互通平台覆盖有线电视网络用户比例超过 95%，乡镇、所有有条件的农村全部实现广播电视数字化，交互式网络电视用户达到 160万户，数字电视用户达到 240万户，其中高清互动电视达到 120万户。 
16.开发新型信息网络服务。大力发展手游、远程教育、网络动漫、互联网影视、虚拟现实等新消费产品和业态，培育新型信息消费产业链。加大对移动互联网新技术、新产品、新模式的应用推广，拓展研发设计、游戏动漫、大数据等业务，向企业和公众提供专业化、个性化、定制化的增值服务。加快推进 “光网 +”行动，促进信息数据与工业、农业、旅游、教育等各产业的融合发展，催生新的消费业态和模式。建立全省统一的政府大数据公共服务平台和政府数据统一开放平台，加快信用、交通、就业、气象、地理等市场服务和民生保障领域的数据开放，促进大数据成为新的消费热点和经济增长点。 (责任单位：省工业和信息化厅、省农业厅、省旅游委、省商务厅、省教育厅、省交通运输厅、省人力资源社会保障厅、省政府金融办 ) 
(五)提供优质教育服务，促进教育消费需求。
积极发展教育产业，不断增加多元化、个性化的优质教育供给，促进教育消费需求。 
17.加快引进优质高等教育资源。高校落户地所在市县要按国家高校设置标准预留专门教育用地给予保障。省政府重点引进的优质高等教育资源由省统筹协调解决用地指标。大力建设三亚荔枝沟教育科技产业园区、海口桂林洋教育科技产业园区，加快完善配套设施和提升公共服务水平。以两大园区为依托，围绕我省十二个重点产业和海洋产业发展需求，引进 10所(个)以上高水平、高层次的研究生院 (研究院 )和中外合作办学机构 (项目)，并在资金、土地、优秀人才及子女落户等方面给予支持。鼓励市县通过政策倾斜，支持学校引进社会力量合作建设。允许公办教育采用政府和社会资本合作 (PPP)模式引入社会〖 JP3〗力量筹资建设。 (责任单位：省教育厅，海口、三亚等市县政府 ) 
18.支持民办学校提供多样化教育服务。规范和促进民办教育发展，放宽社会力量办学准入条件，除国家政策和法律规定外，一律不得限制。重新梳理民办学校准入条件和程序，改进许可方式，简化许可流程，减少事前审批，加强事中事后监管，提高政府管理服务水平。支持社会力量举办非营利性民办学校。 (责任单位：省教育厅，有关市县政府 ) 
19.加快发展终身教育和教育培训服务体系。依托高等学校和职业院校，推动学历教育和职业培训结合，探索建立社区教育、职业教育、成人高等教育、远程教育、教育培训等互为补充、有机结合的终身学习 “立交桥 ”。通过开放大学、老年大学、在线学习、在校学习等多种方式为全民终身学习提供途径和条件，满足社会成员提升文化素质和生活质量的广泛需求。 (责任单位：省教育厅，有关市县政府 ) 
(六)大力发展本土特色文体产业，扩大文体消费需求。
依托资源环境禀赋，重点推动体育、演艺、动漫游戏和影视产业发展，增加特色文体产品供给，满足人民群众日益增长的文体消费需求。
20.全力推进体育产业。加快发展健身休闲产业，重点发展水上运动、骑行、汽车露营、越野、高尔夫等时尚前沿、消费引领的运动项目，打造海南水上运动季品牌活动和海南国际马拉松赛等新赛事。进一步提升海南岛国际公路自行车赛、环海南岛国际大帆船赛、海南高尔夫球公开赛等 “三大品牌赛事 ”的品牌价值，提高和增强观澜湖世界高尔夫球锦标赛、高尔夫球明星赛、中华龙舟大赛等体育赛事影响力，支持在我省举办大型国际体育赛事，向世界宣传推介海南，促进海南体育产业健康快速发展。 (责任单位：省文化广电出版体育厅，各市县政府 ) 
21.加快发展影视产业。打造观澜湖华谊冯小刚电影公社、长影环球 100、儋州海花岛国际影视制作和拍摄基地，支持阿里巴巴文娱集团和爱奇艺创意中心等发展，推动加快形成集策划、拍摄、制作、发行等各环节的影视产业链。 (责任单位：省文化广电出版体育厅、省发展改革委，海口、儋州等市县政府 ) 
22.积极培育演艺产业。挖掘黎苗文化、南洋文化、历史民俗、红色文化等元素，开发一批地域风情浓郁、凸显海南特色的旅游演艺精品项目。推动琼剧会馆建设，推动三亚千古情演艺剧场、槟榔谷演艺剧场等一批传统演艺剧场、演艺形式升级，加快推进三亚海棠湾梦幻不夜城等项目建设，推动旅游演艺发展。 (责任单位：省文化广电出版体育厅，三亚、保亭等市县政府 )
到2020年，文体产品和服务更加丰富，文体消费市场体系更加健全，文体消费环境更加完善，全省文体消费总量、人均文体消费支出和消费满意度均实现有效提升。 
(七)推动创新发展，提升消费市场开放水平。
紧抓我省服务贸易创新发展试点示范机遇，大力发展服务贸易和跨境电商，在扩大开放中挖掘消费市场新需求。 
23.全面推进服务贸易创新发展试点。到 2018年2月，我省要完成旅游、运输、文化服务、保险服务、服务外包、中医药服务等 6大领域 8个方面的创新发展试点任务，在服务贸易管理体制、促进机制、政策体系和监管模式等方面探索取得新成果，形成若干可复制、可推广的经验与做法，初步形成法制化、国际化、便利化的服务贸易双向开放营商环境。建设服务贸易新型网络平台，引导企业承接国际 (离岸 )软件服务外包业务，抢占境外消费市场。 (责任单位：省商务厅，省服务贸易创新发展联席会议成员单位 ) 
24.持续推进跨境电子商务综合示范区建设。鼓励省内商贸企业利用跨境电子商务平台，拓宽商品进出口渠道，做大做强我省特色商品出口业务，加快发展我省跨境电子商务进口业务。 (责任单位：省商务厅、省海防与口岸办、海南出入境检验检疫局、海口海关，有关市县政府 )
三、政策保障措施 
(一)深化制度改革。
健全开放透明的消费市场规则，消除跨地区、跨行业商品和服务消费的体制壁垒。全面清理地方保护政策。归并优化部门审批职能，探索建立市县综合行政审批管理体制，实行综合审批、一枚印章管到底。深化商事制度改革，最大限度简化审批程序，建立企业从设立到退出全过程规范管理制度。加快推进电子证照库建设和“多证合一 ”改革，实行异地办理、全城通办。建立便捷的市场退出机制。禁止对省外企业与外商投资企业及其产品和服务设定歧视性和潜在排斥性准入条件。切实破除行政垄断、行业垄断和地方保护，实现企业自主经营、平等竞争，消费者自由选择、自主消费，旅游等各类商品与要素自由流动、平等交换，营造透明、开放、公平的消费环境。 (责任单位：省工商局、省政府政务服务中心、省编办、省物价局，各市县政府，洋浦经济开发区管委会 ) 
(二)加大财税支持。
加强专项资金整合统筹，集中财力支持旅游等重点产业、产业园区和适应新消费需要的基础设施和公共平台建设。加大对教育、医疗、养老等领域公共服务设施的投资力度。大力推广政府和社会资本合作 (PPP)模式，鼓励和吸引社会资本参与新消费相关基础设施和公共服务领域建设。强化财政资金引导作用，落实创业担保贷款、税费减免、财政贴息、资金补贴等扶持政策。深化国税、地税征管体制改革。按照国家统一部署，推进消费税改革。 (责任单位：省财政厅、省发展改革委、省国税局、省地税局 ) 
(三)创新金融服务。
积极发展创新型非抵押类贷款模式，鼓励金融机构开发适合新消费特点的各类金融产品和服务。完善消费信贷产品体系，加大消费信贷的投放力度。鼓励保险公司与银行合作，支持发展 “政银保 ”模式业务及各类商业贷款保证保险，为小微企业贷款提供风险保障 ;鼓励保险机构开发更多适合旅游、医疗健康、文化体育等消费热点的保险险种 ;支持我省有关机构或企业新设人寿、财产保险公司。规范发展多层次资本市场，推动市场主体规范改制、挂牌上市、发行债券，拓宽直接融资渠道，为培育市场消费主体拓展新领域、新空间。适时设立或引进消费金融公司等新兴金融机构。 (责任单位：省政府金融办、省发展改革委，人行海口中心支行、银监会海南监管局、保监会海南监管局，有关市县政府 ) 
(四)完善用地政策。
按照 “盘活存量、严控增量 ”原则 ,重点保障新消费发展需要的公共服务设施、交通基础设施、市政公用设施等建设用地 ;保障战略性新兴产业、生产性和生活性服务业、科研机构及科技企业孵化机构发展用地 ;多途径保障电动汽车充电设施、移动通信基站等小型配套基础设施用地。优化存量建设用地结构，积极盘活低效利用建设用地。推广在建城市公交站场、大型批发市场、会展和文化中心地上地下立体开发及综合利用。鼓励原用地企业利用存量房产和土地发展研发设计、创业孵化、节能环保、文化创意、健康养老等服务业。对引入的企业总部及企业设立的研发中心用地，可优先办理建设用地审批手续。依法盘活农村建设用地存量，重点保障农村养老、文化及社区综合服务设施建设用地，合理规划现代农业设施建设用地。 (责任单位：省国土资源厅、省交通运输厅、省商务厅、省文化广电出版体育厅、省民政厅、省住房城乡建设厅，各市县政府，洋浦经济开发区管委会 ) 
(五)强化人才支撑。
加大人才培养和引进力度，为消费升级、产业升级、创新发展提供人才保障。加快实施 “海南省十二大重点产业人才培养五年行动计划 ”，紧密对接旅游产业、热带高效特色农业、互联网产业、医疗健康产业、教育文化体育等发展需求，大力发展高等教育、职业教育，培养一批实用型、技能型、创新型和科技领军人才队伍。尽快实施我省顶尖人才团队引进计划，支持高校、企业围绕省重点产业发展需求引进国内外一流的科技团队，继续实施 “好校长、好老师引进工程 ”，组织实施 “南海学者引进计划 ”、“海外名师引进计划 ”、“好院长、好医生引进工程 ”，为重点产业发展提供高水平的智力支持。 (责任单位：省教育厅、省人力资源社会保障厅、省编办、省科技厅、省旅游委、省海洋与渔业厅、省农业厅、省工业和信息化厅 )
各市县、各部门要高度重视并主动顺应消费升级大趋势，积极发挥新消费引领作用，加快培育形成新供给新动力，推动经济实现有质量、有效益、可持续发展。各责任单位要按照省委、省政府确定的各项工作任务目标和时间节点，细化分工、明确责任，抓好落实。要加大督查力度，确保各项任务措施落到实处、取得实效。
