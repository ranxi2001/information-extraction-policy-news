海南省统筹疫情防控和经济恢复提振⾏动⽅案
为全⾯深⼊贯彻落实习近平总书记关于 “疫情要防住、经济要稳住、发展要安全 ”重要指⽰要求，全⼒克服新⼀轮疫情对经济平稳健康发展造成的影响，⾼效精准统筹疫情防控和经济社会发展，多措并举推动全省经济恢复提振，制定本⾏动⽅案。
⼀、⾏动⽬标
坚持 “⼀⼿打伞、⼀⼿⼲活 ”，迅速有序推动全省主要⾏业九⽉份全⾯复⼯复产、第四季度发⼒冲刺，⼒争实现全年经济发展⽬标。供给侧⽅⾯，第⼀产业、第⼆产业⼒保实现全年增长⽬标，⾮接触式服务业应多作贡献，接触式服务业应尽量减少损失。需求侧⽅⾯，固定资产投资全年同⽐ (下同 )增长 10%，社会消费品零售总额全年增长 1%以上，货物进出⼜总额全年增长 25%，服务进出⼜总额和实际利⽤外资全年增长 15%。在本轮疫情中，三沙、⽂昌、屯昌、琼中、保亭、⽩沙等 6个未发⽣疫情的市县要确保实现、⼒争超额完成全年经济发展⽬标；海⼜、五指⼭、澄迈、琼海、定安、昌江等 6个受疫情影响相对⼩的市县要⼒争实现全年经济发展⽬标；儋州、陵⽔、东⽅、临⾼、万宁、乐东等 6个受疫情影响较⼤的市县要全⼒以赴稳定经济增长，尽量减少疫情影响；三亚市要千⽅百计稳定经济运⾏，⼒争实现最好的结果。
⼆、⾼效精准抓好疫情防控 
1.
严格执⾏疫情防控政策。严格实施《新型冠状病毒肺炎防控⽅案 (第九版 )》，因时因势动态调整防控措施，提升各环节⼯作质量和效率。建⽴疫情防控 “九不准 ”督查机制，严格落实分区分级精准防控要求，切实避免过度防控与层层加码。 (牵头单位：省疫情防控⼯作指挥部各有关⼯作组、省疫情防控指挥体系各有关⼯作专班，各市县政府 ) 

2.
加快提升疫情防控能⼒。全⾯强化核酸检测 “采、送、检、报 ”关键环节，提⾼检测质量与效率。按照政府主导、市场参与⽅式提升常态化核酸检测能⼒。推动建设⼀批永久性⽅舱医院，加强可转换⽅舱医院储备。建⽴覆盖全省的 “防疫通 ”系统，编制印发隔离点管理⼿册，多渠道夯实隔离资源储备。提升社会管理信息化平台，推动形成全域全天候全⼈群疫情防控信息化管理闭环。有序推进全⼈群新冠疫苗接种。 (牵头单位：省疫情防控⼯作指挥部各有关⼯作组、省疫情防控指挥体系各有关⼯作专班、省发展改⾰委、省委政法委、省卫⽣健康委 )

三、蹄疾步稳推进海南⾃由贸易港建设 

3.扩⼤ 
“⼀线放开、⼆线管住 ”试点范围。⼤⼒推动海南⾃由贸易港 “⼀线放开、⼆线管住 ”进出⼜政策试点扩区落地，⽀持试点扩区企业适⽤加⼯增值免关税政策。 (牵头单位：省商务厅、海⼜海关 ) 

4.
促进⿎励类产业扩围享惠。加快推动海南⾃由贸易港⿎励类产业⽬录调整优化，进⼀步明确⿎励类产业企业实质性经营标准，⽀持市场主体依规充分享受⿎励类产业优惠政策。 (牵头单位：省发展改⾰委、省税务局 ) 

5.强化 
“零关税 ”政策优势。积极推进交通⼯具及游艇 “零关税 ”政策清单调整，扩展政策享惠主体范围、清单⽬录，优化政策适⽤条件。研究制定适合平⾏进⼜车的国六环保信息公开和认证标准，畅通进⼜车辆采购渠道。 (牵头单位：省商务厅、海⼜海关、省交通运输厅、省⽣态环境厅 ) 

6.
加快推进封关运作项⽬。全⾯推动全岛封关运作项⽬提速建设，确保第⼀批项⽬年内全部开⼯并加快建设。以超常规⼿段加强⼜岸、⾮设关地等基础设施和配套设施建设。 (牵头单位：省发展改⾰委、省商务厅、海⼜海关、海南海事局、省财政厅、省⾃然资源和规划厅、省公安厅 )

四、有⼒有序推动复⼯复产 

7.
建⽴复⼯复产协同机制。分市县建⽴由属地政府和园区主管部门负责，省级⾏业主管部门、疫情防控部门指导的复⼯复产服务保障⼯作机制，及时解决企业⼈员到岗、⽣产资料和防疫物资保障、物流运输等⽅⾯困难问题。分产业制定疫情防控指引。强化重点产业链供应链 “⽩名单 ”企业服务保障机制。常态化排查、取消制约复⼯复产的不合理限制。指导产业园区和有条件的企业制定闭环⽣产预案。 (牵头单位：各市县政府，省级各重点产业主管部门、省卫⽣健康委、省交通运输厅、省⼈⼒资源社会保障厅、省公安厅、省疫情防控指挥部疫情防控组 ) 

8.
畅通省内外交通运输通道。优化提升运输通⾏证办理系统。加强省内应急物资中转接驳站布局和建设。积极争取在琼州海峡北岸设置应急物资临时中转点。持续免费为货车司机开展核酸检测和抗原检测。 (牵头单位：省交通运输厅、省发展改⾰委、省卫⽣健康委，各市县新冠肺炎疫情防控指挥部 ) 

9.启动 
“万名⼲部下企业 ”服务⾏动。⾯向全省 “四上 ”企业、 “专精特新 ”企业、省重点项⽬、省重点园区开展“⼀对⼀ ”助企服务，帮助企业解决突出问题，推动企业迅速复⼯复产，⽀持企业提振发展质量效益。 (牵头单位：省 “万名⼲部下企业 ”服务⾏动领导⼩组成员单位，各市县政府 ) 


五、持续加⼒强化助企纾困 
10.制定实施助企纾困特别措施 (2.0版)。系统评估现有助企纾困措施成效，结合市场主体困难诉求制定出台新⼀批助企纾困升级版措施，有效对冲疫情影响。 (牵头单位：省发展改⾰委 ) 
11.
充分释放现有政策效能。开展省级助企纾困政策全⾯系统梳理，形成政策汇编库、政策解读库和案例库。编制政策解读联系名录，建⽴ “⼀策⼀专⼈ ”解读答疑机制。全⾯系统开展助企纾困政策和惠企政策全媒体宣传⾏动。建设完善海南惠企政策兑现服务系统，确保政策⼀⽹通查和尽知尽享。 (牵头单位：省政府营商环境主管部门、省发展改⾰委、省委宣传部、省⼤数据管理局、省级其他各有关单位 )

六、深挖潜⼒扩⼤有效投资 

12.
加强重点领域投资。完善精准对接服务机制，加快推进 G98环岛⾼速公路⼤三亚段扩容⼯程、环热带⾬林国家公园旅游公路、洋浦疏港⾼速公路等重⼤交通基础设施项⽬前期⼯作，⼒争尽早开⼯。推动海⼜绕城公路美兰机场⾄演丰段⼯程等在建重⼤交通基础设施项⽬提速建设。⼤⼒推进百万吨⼄烯、丙烯腈、⽂昌航天超算中⼼等重⼤产业项⽬，确保如期建成投产。强⼒推动 26个新安排⽤能指标⼯业项⽬开⼯建设。加快开展 6个海上风电⽰范、试验项⽬前期⼯作，⼒促年内全⾯开⼯。落实国际旅游消费中⼼重⼤产业项⽬落地的若⼲措施，推动⽂化旅游、会展等重⼤产业项⽬落地实施。推动省艺术中⼼等公共服务重点项⽬尽快开⼯，加快海南医学院第⼀附属医院江东新院区等在建公共服务项⽬进度。推动海⼜、三亚等市县加快城市燃⽓管道⽼化更新改造。推动⽜路岭灌区⼯程开⼯，加快 26万亩⾼标准农⽥、万宁乌场等 3⼤渔港、南繁种业、美丽乡村等农业农村基础设施建设。 (牵头单位：省发展改⾰委、省交通运输厅、省⼯业和信息化厅、省旅游和⽂化⼴电体育厅、省教育厅、省卫⽣健康委、省住房城乡建设厅、省⽔务厅、省农业农村厅，海南电⽹公司，各市县政府 ) 

13.
夯实重点地区投资。⽀持海⼜市开展重点项⽬冲刺攻坚⼤⽐武，确保 115个重⼤投资项⽬完成年度投资任务，开⼯建设⼀批城市更新项⽬，⼒促海洋⼯程装备合规纳统。⽀持三亚市打好固定资产投资百⽇攻坚战，推动已供地项⽬加快建设、未供地项⽬加快供地。⽀持儋州市聚焦产业链和交通基础设施两⼤领域推动投资提速。投资计划相对年度⽬标有缺⼜的市县必须全⼒补齐缺⼜，没有缺⼜的市县必须⼒保完成任务并⼒争多作贡献。强化市县投资周调度机制，指导市县建⽴ “专项协调、专⼈跟踪 ”项⽬服务机制。 (牵头单位：各市县政府，省发展改⾰委、省交通运输厅、省⾃然资源和规划厅、省住房城乡建设厅 ) 

14.
加快项⽬谋划和前期⼯作。围绕四⼤主导产业、 “五⽹ ”提质升级加强重⼤项⽬谋划储备，搭建⾼质量交通基础设施项⽬库。建⽴项⽬审批绿⾊通道，依法合规最⼤限度缩短审批时间。针对项⽬审批、规划⽤地⽤林等关键环节开展前期⼯作攻坚⾏动，结合 “三区三线 ”划定加强耕地、林地占补指标等空间要素分期分段保障。开通环评绿⾊通道，推动符合法定规划、 “三线⼀单 ”准⼊要求项⽬审核提速。建⽴全省重点⽤能项⽬清单，按需开展动态调整，优先保障优质项⽬⽤能需求。 (牵头单位：省发展改⾰委、省⾃然资源和规划厅、省林业局、省⽣态环境厅、省级其他各有关单位，各市县政府 ) 

15.
⽤⾜⽤好促投资政策⼯具。加快 2022年地⽅政府专项债券⽀出进度。⽤好专项债务结存限额，发⾏新增专项债券⽀持符合条件的项⽬建设。优化推进有效投资重要项⽬协调机制，强化政策性开发性⾦融⼯具项⽬协调对接，提升项⽬审核通过率。积极推进⽜路岭灌区⼯程、 G98环岛⾼速公路⼤三亚段扩容⼯程和洋浦疏港⾼速公路等 PPP项⽬前期⼯作。⽀持符合条件的民间投资项⽬发⾏基础设施不动产投资信托基⾦ (REITs)。⽀持开发性银⾏和政策性银⾏⽤⾜新增贷款额度，引导商业银⾏资⾦扩⼤投资渠道。 (牵头单位：省发展改⾰委、省财政厅、省⽔务厅、省国资委、省交通运输厅、⼈民银⾏海⼜中⼼⽀⾏、银保监会海南监管局，各市县政府 ) 

16.
稳定房地产开发投资。确保今年计划开⼯的 5万套安居房 9⽉底前全⾯开⼯，加快已建成安居房配售。出台加强城市更新规划和⽤地保障的综合性政策。对拥有 1套住房的居民家庭，为改善居住条件再次申请商业性个⼈住房贷款购买普通商品住房的，最低⾸付款⽐例调减为不低于 40%。探索对未设⽴住房公积⾦贷款抵押的⼆⼿房开展 “带押过户 ”登记试点。适当新增住房建设⽤地指标，⽀持 “⼀城⼀策 ”灵活运⽤信贷⼯具，满⾜刚性和改善型住房需求。研究合理调整城市更新商办类项⽬可产权分割最⼩单位建筑⾯积。 (牵头单位：省住房城乡建设厅、省⾃然资源和规划厅、省司法厅、⼈民银⾏海⼜中⼼⽀⾏、银保监会海南监管局，各市县政府 ) 

17.
强化项⽬建设⼟地要素保障。出台加强项⽬建设⽤地要素保障的有关政策，开展规划与要素保障 “百⽇会战”⾏动。修订节约集约⽤地办法和评价标准、海域使⽤权审批出让管理办法。积极推动存量⼟地盘活利⽤，⼒争2022年底批⽽未供⼟地、闲置⼟地处置率分别达到 25%和60%。加快耕地垦造和补充耕地项⽬整改，推动垦造⼯作与 “⼟地超市 ”充分衔接。在新⼀轮国⼟空间规划批复前，超出现⾏规划但符合 “三区三线 ”等国⼟空间规划管控要求的近期建设项⽬，可由所在市县政府附图承诺后，通过预⽀规划规模的⽅式保障⽤地。⽀持分期分段办理⽤地审批。 (牵头单位：省⾃然资源和规划厅，各市县政府、省级重点产业园区管理机构 ) 

18.
激发促投资内⽣动⼒。对年度投资同⽐增速超过全省增速 10个百分点以上、且占全省投资⽐重较上年提 ⾼1个百分点以上的市县，给予不低于年初预安排新增建设⽤地规模 10%(不含预安排给省级重点园区的新增建设⽤地指标 )的新增建设⽤地奖励。在年度市县⾼质量发展综合考核中，提⾼固定资产投资指标权重。按规定对在促投资⼯作中作出突出贡献的市县和个⼈进⾏表彰。 (牵头单位：省发展改⾰委、省⾃然资源和规划厅、省⼈⼒资源社会保障厅，各市县政府 )

七、引促并举激活消费市场 

19.
有序实施消费引流主题活动。待条件允许后有序开展 “旅游消费嘉年华 ”“抗疫英雄游海南 ”“莘莘学⼦畅琼游”等主题活动。待时机成熟时迅速推出 “疫散云开，重见海南 ”旅游宣传推⼴活动，策划举办超 100场旅游重启宣传推⼴活动，以吸引游客回流带动消费全⾯回升。 (牵头单位：省旅游和⽂化⼴电体育厅 ) 

20.
聚焦重点领域激发消费潜⼒。举办⾸届海南离岛免税购物节专场营销活动，组织开展家电促销活动，开展新⼀轮新能源汽车下乡活动。按规定延续实施免征新能源汽车购置税政策⾄ 2023年底。全⾯取消符合 “国五 ”排放标准⼩型⾮营运⼆⼿车的迁⼊限制。 (牵头单位：省商务厅、省财政厅、省公安厅、省税务局、省⼯业和信息化厅 )
