发布日期 :2018-05-15 
各地 (市)行署 (人民政府 ),自治区各委、办、厅、局 :
《2018年西藏自治区打击侵犯知识产权和制售假冒伪劣商品工作要点》已经自治区人民政府同意 ,现印发给你们 ,请认真贯彻执行。
西藏自治区人民政府办公厅 
2018年4月24日 
2018年西藏自治区打击侵犯知识产权和制售假冒伪劣商品工作要点 
2018年,自治区打击侵犯知识产权和制售假冒伪劣商品工作 (以下简称打击侵权假冒工作 )要以习近平新时代中国特色社会主义思想为行动指南 ,深入学习贯彻党的十九大精神 ,落实党中央、国务院和自治区党委、政府各项决策部署 ,坚持以人民为中心的发展思想 ,深入贯彻落实《国务院关于新形势下加强打击侵犯知识产权和制售假冒伪劣商品工作的意见》 (国发〔 2017〕14号)和全国打击侵权假冒工作领导小组办公室《关于贯彻落实〈国务院关于新形势下加强打击侵犯知识产权和制售假冒伪劣商品工作的意见〉任务分工的通知》 (打假办发〔 2018〕2号)精神 ,按照高质量发展要求 ,坚持稳中求进、依法治理、打建结合、统筹协作、社会共治的原则 ,下沉工作重心 ,聚焦重点区域、重点领域、重点环节、重点产品 ,严厉打击假冒伪劣滋生蔓延势头 ,不断优化营商环境 ,切实保护消费者、权利人合法权益 ,为满足人民日益增长的美好生活需要提供有力保障。
一、狠抓工作补齐短板 
(一)加强 “两法衔接 ”工作。大力推进行政执法和刑事司法衔接 (以下称 “两法衔接 ”)信息系统建设。各地 (市)、县(区)要结合各自实际 ,按照 “一级事权 ,一级财权 ”的原则 ,加快 “两法衔接 ”信息系统建设 ,将“两法衔接 ”信息系统建设资金和运行维护费用纳入本级财政预算。推进执法部门工作平台与自治区 “两法衔接 ”信息系统有效对接 ,实现信息共享 ,情报互通。组织开展自治区、地 (市)、县 (区)三级 “两法衔接 ”业务培训工作。 (各级打击侵权假冒工作领导小组办公室、各级财政部门牵头 ,各成员单位按职责分工负责 )不断完善行政执法部门和刑事司法部门信息共享、案情通报、案件移送制度。规范案件信息管理和应用 ,推动打击侵权假冒涉案物品检验鉴定和保管处理体系建设 ,提高案件移送后按规定流转的比率 ,促进案件移送程序规范化。 (打击侵权假冒工作领导小组办公室、区检察院牵头 ,各成员单位按职责分工负责 ) 
(二)建立健全工作机制。完善会议、沟通和通报机制 ,每年至少召开 2次打击侵权假冒工作领导小组联席会议。建立健全完善跨区域跨部门联合执法机制 ,加强互通互联和协作配合 ,形成上下联动、部门联动、区域联动 ,横向到边、纵向到底的网格化打击侵权假冒工作格局。 (各级打击侵权假冒工作领导小组办公室牵头 ,各成员单位按职责分工负责 )深化 “丝绸之路 ”经济带沿线 10省(区、市 )联席会议制度 ,落实线索通报、案件流转、执法联动、检验鉴定结果互认等机制。 (打击侵权假冒工作领导小组办公室牵头 ,各成员单位按职责分工负责 ) 
(三)加强信息报送工作。建立和完善打击侵权假冒工作信息报送和数据统计制度 ,规范信息采编、报送、统计工作。以政府网站为主渠道 ,推进信息公开 ,做到应公开尽公开。落实信息公开督查考核制度。 (打击侵权假冒工作领导小组办公室牵头 ,各成员单位配合 ) 
(四)强化文化行政执法。加强版权执法监管 ,抓好案件查办 ,开展系列专项行动。加大版权案件查处力度 ,在版权办案数量上有所突破 ,强化案件督办 ,加大犯罪案件 (线索 )移送力度。适时开展版权执法培训工作。 (新闻出版广电局牵头 ,文化市场综合执法总队按职责分工负责 ) 
(五)组织实施无害化销毁。加大侵权假冒商品环境无害化销毁工作力度 ,确保将罚没和收缴的侵权假冒商品移交给有资质的单位进行无害化处理。定期更新并公布本地具有环境无害化销毁能力的单位名录 ,对侵权假冒商品免费予以销毁。避免采用焚烧、简单填埋等方式 ,防止二次污染。 (环境保护厅牵头 ,各成员单位按职责分工负责)组织开展侵权盗版及非法出版物销毁活动。 (环境保护厅牵头 ,新闻出版广电局、文化市场综合执法总队按职责分工负责 )
二、深入开展重点领域治理 
(一)深化互联网领域侵权假冒治理。加强网络市场监管 ,开展 2018网络市场监管专项行动 ,强化对网上销售侵权假冒商品违法行为的追踪溯源和全链条打击 ,推进 “网络市场监管与服务示范区 ”创建工作 ,深入推进电子商务领域产品质量执法 ,加大线索摸排、重点时段抽查和打击力度。促进和规范互联网医疗服务发展。打击 “假海淘 ”“假代购 ”,督促寄递企业严格执行收寄验视、实名收寄、过机安检三项制度 ,严肃查处寄递侵权假冒商品违法行为。 (通信管理局、工商局、区党委网信办、工业和信息化厅、公安厅、农牧厅、卫生计生委、拉萨海关、质监局、食品药品监管局、林业厅、邮政管理局按职责分工负责 )打击网络侵权盗版 ,组织开展 “剑网 2018”专项行动 ,加大网络影视、音乐、新闻、文学、动漫、游戏、 APP、电子商务、网盘、网络广告联盟等领域版权监管。强化对大型商业网站的重点监管 ,扩大监管范围和监管对象 ,开展版权保护预警工作 ,加强对区内优秀影视作品等专项保护,加强媒体融合背景下新闻出版版权保护。以网络表演、网络游戏、网络音乐、网络动漫等为重点 ,组织查处违法违规互联网文化产品和经营单位。深入开展电子商务领域专利执法维权。 (通信管理局、区党委网信办、公安厅、工业和信息化厅、文化厅、工商局、新闻出版广电局、知识产权局按职责分工负责 )强化互联网企业监管。继续加强网站备案、网际协议地址 (IP地址 )、域名等基础管理 ,严格网络零售第三方平台交易规则备案管理 ,指导和督促互联网企业切实落实主体责任 ,建立完善长效治理机制。加快网络信息内容执法体系建设 ,全面清理整治网上侵权假冒违法违规信息。 (通信管理局、区党委网信办、工业和信息化厅、公安厅、商务厅、工商局、质监局、新闻出版广电局、文化市场综合执法总队按职责分工负责 ) 
(二)加强农牧区和城乡结合部市场治理。从生产源头、流通渠道、消费终端等环节开展集中专项整治 ,突出市场监管执法 ,重点打击违法犯罪 ,维护农牧区和城乡结合部 ,特别是边远地区市场秩序。 (公安厅、农牧厅、商务厅、工商局、质监局、食品药品监管局、林业厅、知识产权局、邮政管理局按职责分工负责 )严厉打击制售假冒伪劣农资违法犯罪行为。部署开展 “绿剑护农 ”“红盾护农 ”农资打假专项行动 ,在春耕和秋冬时节 ,重点整治农药隐性添加、兽药非法添加、肥料有效成分不足、种子无证套牌等严重危害农产品质量安全和侵害农牧民利益的问题,整治农资市场秩序 ,净化农资市场环境 ,加强农业投入品源头管控。 (农牧厅、公安厅、工商局、质监局按职责分工负责 )开展林木种苗质量抽查 ,对不低于 3%的持有林木种子经营许可证企业开展 “双随机 ”抽查。积极开展林木种苗打假、查办侵犯植物新品种权案件 ,提高查处案件数量。 (林业厅按职责负责 ) 
(三)强化进出口环节侵权假冒治理。大力开展 “丝路清风行动 ”,持续开展进出口商品侵权打击行动。深入落实“丝绸之路 ”沿线 10省(区、市 )联席会议机制 ,加强合作 ,推动 “丝路清风 ”行动 ,维护 “中国制造 ”海外形象。 (拉萨海关牵头 ,工商局、工业和信息化厅、公安厅、农牧厅、商务厅、质监局、食品药品监管局、知识产权局、区党委网信办、邮政管理局按职责分工负责 ) 
(四)加大涉税案件查处力度。加强侵权假冒涉税案件线索移交和调查处理工作 ,严肃查处相关税收违法行为。继续组织开展相关行业和领域税收专项整治 ,加强对侵权假冒市场主体日常税收监管。 (国税局牵头 ,各成员单位按职责分工负责 )
三、不断强化商品质量监管 
(一)严格重点产品管理。围绕事关生命健康、财产安全、环境保护的重点产品 ,深入开展 “质检利剑 ”专项行动,对强制性产品认证目录内产品未经认证违法出厂销售、进口或在经营活动中使用等行为加大查处力度 ,严肃查处质量违法行为。 (质监局按职责负责 )进一步强化流通领域商品质量监管执法 ,开展 “红盾质量维权行动 ”。加强重点领域广告监管 ,严肃查处虚假违法广告。 (工商局按职责负责 )继续严厉打击食品药品违法犯罪行为 ,注重案源线索发掘和研判 ,聚焦重点领域、重点环节、重点品牌 ,加大大案要案查处力度。 (食品药品监管局按职责负责 )组织开展消毒产品监督抽检工作 ,及时向社会通报违法企业和不合格产品名单 ,加强大案要案督办。 (卫生计生委按职责负责 )严格车用燃油监管 ,重点加强车用汽柴油和普通柴油质量监管 ,集中力量严肃查处生产、销售假冒伪劣车用油品行为 ,严厉打击黑加油站点。对非标油进行综合治理 ,加大道路运输环节稽查抽检力度。 (公安厅、环境保护厅、商务厅、国资委、国税局、工商局、质监局按职责分工负责 ) 
(二)促进产品质量提升。开展全区知名品牌和出口食品农产品质量安全示范区建设工作 ,推进出口商品打击侵权假冒工作 ,对重点区域开展集中整治 ,促进区域产品质量提升。 (质监局、拉萨海关按职责分工负责 )
四、从严查处侵犯知识产权行为 
(一)依法查处专利侵权假冒行为。深入开展执法维权 “雷霆 ”专项行动 ,针对电子商务、食品药品、环境保护、安全生产、高新技术等重点领域 ,以及展会、进出口等重点环节 ,加大对专利侵权假冒行为的打击力度 ,完善举报投诉快速反应机制 ,加强对境内大型展会、重大活动的维权援助服务。 (知识产权局牵头 ,区检察院、区高法院、公安厅、商务厅、卫生计生委、工商局、质监局、食品药品监管局、拉萨海关按职责分工负责 ) 
(二)持续加强商标专用权保护。严肃查处商标恶意抢注行为 ,加大对驰名商标、地理标志、涉外商标、老字号商标等的保护力度 ,重点打击商标侵权行为 ,严肃查处商标代理机构不正当竞争等违法行为 ,规范代理行业秩序。加强打击侵犯商业秘密工作部署 ,落实查处重大仿冒侵权案件协调机制 ,加大对侵权商业秘密行为的查处力度。 (工商局按职责负责 )实施出口知识产权优势企业培育和塑造计划 ,为企业创新发展和 “走出去 ”提供支持。 (知识产权局、商务厅、拉萨海关按职责分工负责 ) 
(三)加大版权保护工作力度。严肃查处侵权盗版行为 ,集中查处一批大案要案 ,加强重点产业专项保护。开展“秋风 2018”专项行动 ,持续打击线上线下侵权盗版活动 ,重点打击盗印重大题材出版物、畅销图书、少儿类图书、资格考试辅导书、教材教辅书的行为 ,持续治理学校内部及周边复印店的侵权盗版现象。加强印刷复制发行行业监管 ,打击印制非法出版物产品行为 ,规范出版物发行市场 ,落实常态化督查机制。 (新闻出版广电局、文化市场综合执法总队按职责分工负责 )开展文化市场暗访抽查。 (文化厅、文化市场综合执法总队按职责分工负责 )严肃查处各类标准侵权盗版行为。 (质监局按职责负责 ) 
(四)巩固扩大软件正版化成果。抓好计算机软硬件采购源头和考核评议工作 ,巩固政府机关、国有企业软件正版化成果。加快推进自治区企事业单位软件正版化工作。强化日常监管 ,加大督查力度 ,组织开展 “回头看 ”。巩固国产软件应用试点成果 ,适度扩大试点范围。推进软件正版化与信息化建设相融合 ,防范重要信息系统和基础设施建设信息安全风险。 (新闻出版广电局牵头 ,工业和信息化厅、财政厅、国资委、工商局、知识产权局按职责分工负责 ) 
(五)加强植物新品种保护。组织打击制售侵权假冒种子种苗工作 ,开展打击农、林业植物新品种侵权行政执法活动 ,加强农业植物新品种测试体系建设 ,进一步提高植物新品种测试能力和侵权证据鉴定水平。 (农牧厅、林业厅按职责分工负责 )
五、充分发挥刑事司法作用 
(一)加大刑事打击力度。完善情报导侦 ,强化线索研判 ,提高打击效能 ,以数据交互为纽带 ,推动一体化作战格局。推进 “云端 2018”打击风险经济犯罪专项行动 ,落实 “春雷行动 ”,打击侵犯知识产权犯罪。发起打击制售假酒犯罪行动 ,聚焦民生、涉外、公共安全领域 ,围绕重大、恶性案件 ,组织专案打击行动 ,提升破案数量。承担全国集群战役线索研判和线索提供任务 ,承办公安部督办案件。 (公安厅牵头 ,各成员单位按职责分工负责 ) 
(二)全面履行检察职能。重点办理一批情节严重、影响恶劣的侵权假冒犯罪案件 ,加强对重点案件、新型案件的研究和督办 ,适时公布打击侵权假冒典型案例 ,监督立案一批人民群众反映强烈、社会影响恶劣的案件。 (区检察院按职责负责 ) 
(三)深入推进司法改革。完善知识产权审判体制机制 ,扎实推进民事、刑事、行政案件审判 “三合一 ”改革 ,总结知识产权法院建设经验 ,推动知识产权法庭建设 ,探索知识产权案件审理机制。加大知识产权刑事司法保护力度 ,依法严厉制裁侵权假冒犯罪行为。加大审判公开力度 ,进一步推进裁判书、审判流程和庭审公开。 (区高法院按职责负责 ) 
(四)加强知识产权司法服务。深入开展为市场主体提供知识产权保护专项法律服务活动 ,培育知识产权保护律师服务示范机构 ,推进知识产权公证服务平台建设。 (司法厅按职责负责 )推进知识产权纠纷仲裁调解试点和知识产权保护工作。 (区高法院、知识产权局按职责分工负责 )
六、不断提升工作能力和水平 
(一)强化执法队伍建设。开展业务培训 ,加强业务指导 ,打造专业队伍 ,提升执法人员业务水平和执法行政能力。 (公安厅、农牧厅、文化厅、拉萨海关、工商局、质监局、新闻出版广电局、食品药品监管局、林业厅、知识产权局、文化市场综合执法总队按职责分工负责 )落实打击侵权假冒工作经费保障。 (财政厅按职责负责 ) 
(二)提升监管信息化水平。充分利用互联网、物联网、大数据、云计算等现代信息技术 ,创新监管方式 ,加强技术监测平台建设 ,探索 “互联网 +监管 ”模式 ,不断提升侵权假冒违法线索发现、收集、甄别、追溯和处理能力 ,提高监管的科学性、有效性。完善统计制度 ,加强执法数据统计应用 ,加强形势分析。 (打击侵权假冒工作领导小组办公室、区党委网信办、工业和信息化厅、公安厅、农牧厅、商务厅、文化厅、拉萨海关、工商局、质监局、新闻出版广电局、食品药品监管局、林业厅、知识产权局、邮政管理局按职责分工负责 )推广知识产权海关保护备案移动执法查询系统 ,上线知识产权海关保护执法系统。推进 12365系统全面上线应用。加快实名收寄信息系统推广应用。 (拉萨海关、质监局、邮政管理局按职责分工负责 )
七、构建社会共治格局 
(一)落实联合惩戒机制。继续推进全区公共信用信息共享平台建设 ,推动深度共享 ,加快 “信用中国 (西藏 )”网站深化服务。加快联合惩戒和红黑名单制度建设 ,推动相关联合惩戒备忘录落地生效。加快地方和城市信用体系建设,推动解决侵权假冒领域严重失信问题。 (发展改革委、人行拉萨中心支行牵头 ,各成员单位按职责分工负责 )建立更加高效便捷可靠的涉企信息归集共享工作体制机制 ,进一步推进侵权假冒失信企业联合惩戒工作。 (工商局牵头,各成员单位按职责分工负责 )实施农资领域违法失信黑名单制度 ,全力推进农资和农产品生产经营主体信用档案建设。 (农牧厅按职责负责 )推进商务领域、寄递行业、林木种苗企业诚信建设工作。 (商务厅、林业厅、邮政管理局按职责分工负责 ) 
(二)落实社会评价机制。将打击侵权假冒违法犯罪活动纳入综治工作 (平安建设 )考核评价体系 ,将打击侵权假冒工作情况纳入群众安全感调查和明察暗访范围 ,加强地方绩效考核 ,全面推进基层督查考核 ,落实各级政府责任。指导各级综治部门对侵权假冒违法犯罪活动突出的地区和单位通过通报、约谈、挂牌督办等方式督促其限期整改并落实 ;对因重视不够、整治无效、防范打击措施不落实而导致违法犯罪现象严重或者发生特大案件 (事)件的地(市),按照有关规定实行一票否决制 ,追究有关领导干部的责任。 (综治办牵头 ,各成员单位按职责分工负责 ) 
(三)探索社会力量参与。支持社会组织明确权责、依法自治、发挥作用 ,开展行业研究、权益维护、信用评价等工作。鼓励第三方机构参与打击侵权假冒工作 ,探索与民间组织、协会、商会、第三方建立联合打击侵权假冒工作机制 ,发挥社会力量共同参与 ,共同营造良好市场环境 ,维护消费者权益。 (打击侵权假冒工作领导小组办公室牵头 ,各成员单位按职责分工负责 )探索政府部门与电子商务平台合作 ,借助大数据资源 ,提升数据分析、风险防范、线索发现和精准打击能力。督促权利人企业落实主体责任 ,强化自我管理和主动维权 ,提升打击侵权假冒工作效率。 (打击侵权假冒工作领导小组办公室牵头 ,公安厅、农牧厅、商务厅、文化厅、拉萨海关、工商局、质监局、新闻出版广电局、食品药品监管局、林业厅、知识产权局按职责分工负责 ) 
(四)大力加强宣传教育。通过多种形式深入宣传打击侵权假冒政策措施 ,展示打击侵权假冒工作及专项行动进展成效 ,统筹做好对内、对外宣传 ,加强舆情监控 ,为深入推进打击侵权假冒工作营造浓厚的舆论氛围。落实 “谁执法谁普法 ”的普法责任制 ,强化执法宣传教育 ,将打击侵权假冒工作法治宣传纳入普法责任制清单。加强新媒体、新技术运用 ,解读相关法律法规 ,曝光典型案例 ,结合 “3·15”国际消费者权益日、 “4·26”知识产权日、 “5·15”打击和防范经济犯罪宣传日等重要节点做好集中宣传。 (区党委宣传部、司法厅、工商局、知识产权局、区党委网信办、商务厅等按职责分工负责 )开展 “诚信兴商宣传月 ”等诚信建设主体宣传工作 ,营造尊重创新、崇法守信的社会氛围。 (发展改革委、人行拉萨中心支行牵头 ,各成员单位按职责分工负责 )推动创新知识产权人才培养机制 ,将保护知识产权等内容纳入中小学课程和高等院校就业创业指导课程 ,培养尊重创造、崇尚创新的意识。 (教育厅牵头,知识产权局按职责分工负责 )
八、深入开展涉外交流合作 
(一)深化国际执法协作。充分利用国际刑警组织和中尼双边机制 ,深入推进国际刑事执法合作 ,策划重点侵权假冒案件跨国联合执法行动。 (公安厅按职责负责 )不断深化国际海关执法合作 ,提升知识产权海关保护国际规则制定的话语权。深化中尼海关执法合作 ,适时开展联合执法行动。 (拉萨海关按职责负责 )加大与各出口国合作力度,加强对可疑检验检疫证和原产地证书审核工作 ,打击伪造检验检疫证书和食品原产地证书违法行为。 (质监局、拉萨海关按职责分工负责 ) 
(二)探索境外防侵权工作机制。完善知识产权综合服务机制 ,帮助企业加快适应国际规则 ,增强知识产权保护意识 ,提升管理知识产权和应对纠纷能力。探索建立境外维权与境内维权工作的联动机制 ,为企业 “走出去 ”服务。 (商务厅、知识产权局按职责分工负责 )拓展海关对外合作 ,协助企业维护海外知识产权利益。 (拉萨海关按职责负责 )建立和完善企业商标海外维权协调机制 ,指导和帮助区内知名企业海外维权。 (工商局按职责负责 ) 
(三)加强外商投资企业知识产权保护。完善外商投资企业知识产权保护 ,着力解决商业秘密保护、商标恶意抢注、 “傍名牌 ”等外商投资企业普遍关心的问题。 (商务厅、知识产权局、工商局牵头 ,各成员单位按职责分工负责)
